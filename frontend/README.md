# AppBI Frontend

Next.js-based frontend for the AppBI Business Intelligence tool.

## Quick Start

1. **Install dependencies**:
   ```powershell
   npm install
   ```

2. **Setup environment**:
   ```powershell
   cp .env.local.example .env.local
   ```

3. **Start development server**:
   ```powershell
   npm run dev
   ```

Visit http://localhost:3000 to access the application.

## Build for Production

```powershell
npm run build
npm start
```
